# Hanami Go to Spec extension for VSCODE

Jump between code and spec in Hanami projects.
Based on the [Rails Go To spec](https://github.com/sporto/rails-go-to-spec-vscode) extension

## Default keybinding:

- Ctrl + Shift + y
- Cmd + Shift + y (Mac)
